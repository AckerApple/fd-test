# fd-test
Fullfillment of required entry exam

## Installation

Checkout the project
```
$ git clone https://github.com/AckerApple/fd-test.git
```

Move into project folder
```
$ cd fd-test
```

Install dependencies
```
$ npm install
```

Run server
```
$ npm start
```

Test all
```
$ npm test
```

Server Development
```
$ npm run watch
```

Test Development
```
$ npm run test:watch
```
